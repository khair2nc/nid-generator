# 🆔 Digital NID Card Generator

একটি সহজ এবং সুন্দর **Bangladesh Digital NID Card Generator** প্রজেক্ট। এটি ব্যবহার করে ব্যবহারকারী তাদের মৌলিক তথ্য এবং ছবি ইনপুট দিয়ে তাৎক্ষণিকভাবে একটি ডেমো NID কার্ড তৈরি করতে পারবেন।

## ✨ বৈশিষ্ট্যসমূহ (Features)
- 📝 **সহজ ইনপুট:** নাম, পিতার নাম, মাতার নাম এবং জন্ম তারিখ ইনপুট দেওয়ার ব্যবস্থা।
- 🖼️ **ছবি আপলোড:** নিজের ছবি আপলোড করে কার্ডে যুক্ত করার সুবিধা।
- 🎨 **রেসপনসিভ ডিজাইন:** কার্ডটি দেখতে বাংলাদেশি আসল স্মার্ট কার্ডের কালার প্যালেটের কাছাকাছি।
- ⚡ **ইনস্ট্যান্ট জেনারেশন:** কোনো পেজ রিফ্রেশ ছাড়াই সাথে সাথে কার্ড তৈরি হয়।
